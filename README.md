# Pizza Sales Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on pizza sales data to discover sales patterns and customer preferences.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data cleaning
- Data merging
- Revenue calculation
- Best-selling pizzas
- Monthly sales trends
- Sales performance analysis

## Key Insights
- Identified the top-selling pizzas
- Analyzed monthly sales patterns
- Found peak sales periods
